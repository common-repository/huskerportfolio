== Description ==

HuskerPortfolio allows to create a unique page for displaying portfolio items with screenshots and additional information such as title, description etc. Also it allows to add additional screenhots (many additional screenshots per 1 portfolio item). You can manage your Portfolio by creating suitable categories related to your work (Like Web Design, Web Development).

== Installation ==

1. Upload huskerPortfolio folder to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Now you are done. Create a page and add shortcode [huskerPortfolio].

== Frequently Asked Questions ==

= HOW TO USE PLUGIN? =

1. Choose Portfolio Categories from the Portfolio menu and fill out your page. Set necessary values for the Name and Slug(Optional),select Parent item(Optional) and save it.
2. Choose Add New Item from the Portfolio menu and fill out Title and description like post. Select Portfolio Category that you have created and lastly upload images from PortFolio Images and publish the post.
3. Your Portfolio will be available on Portfolio page.
